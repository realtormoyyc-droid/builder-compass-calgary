# Calgary Builder Compass

**The Ultimate Builder Intelligence Platform for Calgary & Surrounding Areas**

A premium web-based tool for home buyers, investors, and realtors to compare 25+ Calgary builders with detailed ratings, customer reviews, investment outlooks, trend analysis, and current incentive tracking.

## Features

✅ **25+ Calgary Builders** with comprehensive profiles
✅ **Strict Scoring System** (Trust Score 79-94, 8-point rating breakdown)
✅ **Customer Reviews & Common Issues** - real buyer feedback and pain points
✅ **5-10 Year Investment Outlooks** for communities
✅ **Current Incentives Hub** - all active promotions updated April 2026
✅ **Market Trend Analysis** - incentive patterns, prefab revolution, green building premium
✅ **Contact Integration** - phone, website, email for each builder
✅ **Advanced Filtering** - by home type, price, rating, area
✅ **Side-by-Side Comparison** - up to 3 builders at a time
✅ **Mobile Responsive** - works on all devices

## Quick Start

### Local Development
1. Download `index.html`
2. Open in any modern browser
3. No server or build process required - pure HTML/CSS/JavaScript

### Vercel Deployment

#### Option 1: Connect GitHub (Recommended)

1. **Create a GitHub Repository**
   - Go to github.com/new
   - Create repo: `calgary-builder-compass`
   - Clone locally: `git clone https://github.com/YOUR_USERNAME/calgary-builder-compass.git`
   - Add files: `cp index.html README.md into your local repo`
   - Commit: `git add . && git commit -m "Initial commit"`
   - Push: `git push -u origin main`

2. **Connect to Vercel**
   - Visit vercel.com and log in
   - Click "New Project"
   - Select "Import Git Repository"
   - Choose your GitHub repo
   - Click "Deploy"
   - Your site will be live at `calgary-builder-compass.vercel.app`

#### Option 2: Direct Upload
- Visit vercel.com
- Click "New Project"
- Click "Drag and drop your code"
- Select `index.html`
- Deploy automatically
